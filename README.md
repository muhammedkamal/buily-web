# Buily 
## Market place Website designed based on MVC and Distributed Database principals
#  System Requirement
- PHP >= 7.1.3
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension
- Ctype PHP Extension
- JSON PHP Extension
-  Composer
# Setup
- composer install
- php artisan migrate --seed
- npm install
- npm run dev
- php artisan storage:link
